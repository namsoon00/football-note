# 태오의 노트 (Football Note) 기획 문서

최종 갱신: 2026-03-03

## 1) 제품 개요
- 제품명: 태오의 노트
- 한줄 설명: 축구 훈련 기록, 성장 통계, 백업/복원, 뉴스, 미니게임을 하나의 앱에서 제공하는 성장 노트
- 핵심 가치:
- 훈련을 기록하고 되돌아보기 쉽게 만든다.
- 성장(시간/빈도/리프팅/신체/평균 비교)을 수치와 코칭으로 이해시킨다.
- 데이터 유실 없이 다른 기기에서도 동일하게 이어서 쓸 수 있게 한다.

## 2) 대상 사용자
- 1차 사용자: 유소년 축구 선수/보호자
- 2차 사용자: 개인 코치, 아마추어 축구 학습자

## 3) 핵심 기능 범위 (현재 구현 기준)
- 온보딩/시작 화면
- 훈련 일지
- 일지 작성/수정/자동저장
- 리스트형/카드형 조회
- 상태 아이콘 기반 컨디션 표시
- 캘린더
- 월간 캘린더 + 날짜별 일지/계획 조회
- 훈련 계획(추가/수정/삭제)
- 날짜 선택 후 기록 추가 시 선택 날짜를 기본값으로 사용
- 일지 항목 스와이프 삭제(삭제 확인 포함)
- 통계
- 기간 선택 기반 통계
- 성장 코치 코멘트
- 성장 그래프(일 단위)
- 평균 비교(키/몸무게/리프팅/훈련량)
- 뉴스
- 채널 기반 피드 조회
- 한국어 환경에서 제목 번역 옵션
- 기사 인앱 브라우저 열기
- 게임
- 2명 공격수 패스/회피/슈팅 흐름
- 난이도, 레벨, 랭킹 기록/조회
- 조이스틱/패스 패드 입력
- 프로필
- 사진/이름/생년월일/축구 시작일/신체 정보
- 백업/복원
- Google Drive 백업/복원
- 로컬 안전 백업 복원
- 설정
- 일반/기본값/뉴스 필터/알림/백업 등 섹션화

## 4) 주요 사용자 흐름
- 훈련 입력 흐름:
- 홈(일지/캘린더) → 기록 추가 → 입력/자동저장 → 목록/캘린더 반영 → 백업(로그인 시)
- 캘린더 중심 흐름:
- 날짜 선택 → 해당 일지/계획 확인 → 수정 또는 스와이프 삭제
- 성장 점검 흐름:
- 통계 탭 → 기간 선택 → 코칭/그래프/평균 비교 확인
- 데이터 연속성 흐름:
- 기기 A 백업 → 기기 B 복원 → 동일 데이터 확인

## 5) IA (탭 기준)
- 일지 탭: 훈련 기록 관리 중심
- 캘린더 탭: 날짜별 계획/실행 관리 중심
- 통계 탭: 성장 분석/평균 비교 중심
- 뉴스 탭: 축구 정보 소비/리텐션
- 게임 탭: 공간/속도/타이밍 학습

## 6) UX 방향
- 한 화면에 너무 많은 정보를 쌓지 않고, 핵심 행동을 상단/헤더 근처에 배치
- 다크/라이트 모두 가독성 확보
- 버튼/선택 박스 상태(활성/비활성) 명확화
- 모바일(특히 iPhone) 터치 조작 우선

## 7) 비기능 요구
- 로컬 우선 저장 후 비동기 백업
- iOS/Android/Web 멀티 플랫폼 동작
- 오프라인 상황에서도 기록 작성 가능
- 에러 발생 시 데이터 손실 최소화

## 8) 향후 확장 아이디어
- 코치/보호자 리포트 공유
- 주간 목표 자동 추천
- 팀 단위 랭킹/챌린지
- 뉴스 개인화 추천
